# Flutter Ollama AI

Projeto em Flutter para demonstrar a utilização dos modelos de LLM da Ollama AI.

- Version: Flutter 3.22.0

### Run
1. Clone o repositório
2. Execute `flutter pub get`
3. Execute `flutter run`
